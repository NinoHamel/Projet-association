Pour faire marcher notre site, il faut bien évidemment installer la base de données qui est dans le dossier de rendu, il faut aussi avoir installé php.
Le site est hébergé à l'adrese suivante : https://cygames.000webhostapp.com/index.php, en revanche, sur la version hébergée, il affiche des warnings
alors que nous les avons désactivés.
Pour la page administrateur, elle n'est pas accesible sans toucher à l'url du site, il faut rechercher : http://localhost:3000/admin.php ( où un autre 
port si votre php n'est pas défini sur le port 3000.

Un compte administrateur est mis à votre disposition si vous souhaitez tester toutes les fonctionnalités :  modifications ou suppresion d'évènements, 
de produits, suppression / changement de rôle d'un utilisateur, changer l'argent disponible pour l'association. Le rôle ancien membre est inutile car
il n'offre pas de fonctionnalité spéciale sur notre site, c'est simplement une distinction qui se verra dans la fiche utilisateur.

Vous pouvez changer votre mot de passe et votre photo de profil dans cette fiche utilisateur, accessible depuis la page d'accueil. Vous pouvez aussi bien
évidemment accéder à la fiche utilisateur d'une autre personne en entrant son pseudo dans l'url, cependant, vous n'aurez accès qu'à ses informations 
publiques.

Le reste des fonctionnalités semble évident à utiliser mais est tout de même précisé dans le II du rapport si vous souhaitez vérifier quelque-chose.
